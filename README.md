# Постфиксное выражение на абстрактной машине

Индивидуальная работа №2 по дисциплине «Языки программирования», вариант 4.

## Задача

Абстрактная вычислительная машина с одним регистром и шестью инструкциями
(LD, ST, AD, SB, ML, DV). На вход даётся выражение в постфиксной записи из
однобуквенных операндов и операций `+ - * /`. Программа печатает инструкции
машины, которые вычисляют это выражение. Результат остаётся в регистре,
промежуточные значения хранятся во временных переменных Tn.

Пример: для `ABC*+DE-/` получаем

```
LD  B
ML  C
ST  T1
LD  A
AD  T1
ST  T1
LD  D
SB  E
ST  T2
LD  T1
DV  T2
```

## Файлы

- `main.py` — меню, точка входа
- `stack.py` — стек на односвязном списке (Node + Stack)
- `checker.py` — проверки файла и корректности выражения
- `Machine4.py` — само решение
- `input.txt` — пример входных данных

## Запуск

```
python main.py
```

Ввод можно дать из `input.txt` или вручную с клавиатуры. Результат
выводится на экран и пишется в `output.txt`.
